# Loom Script Generator

Genera el script de un video Loom de 2–3 minutos personalizado para un lead específico. El video muestra su negocio vs competidores en Google Maps — es el paso 4 del flujo de ventas (día 6 sin respuesta).

## Inputs necesarios
```
business_name, owner_name (si lo tienes), city, niche,
rating, reviews, website (o NO WEBSITE),
competitor_1, competitor_2 (de Google Maps búsqueda manual),
recommended_service, outreach_language
```

```
"Hey [nombre], soy Keyner de MultiVenza Digital.
Grabé este video específicamente para [business_name]
porque vi algo en Google que creo que te va a interesar."
```

## Notas para la grabación

**Setup técnico:**
- Grabar pantalla + cara (esquina inferior derecha)
- Fondo limpio o logo MultiVenza
- Audio con auriculares — nada de eco
- Duración máxima: 3 minutos

**Durante la grabación:**
- Buscar el negocio en Google Maps EN VIVO (no captura)
- Hablar despacio, con confianza
- El nombre del negocio al menos 2 veces
- Si es en español: más cálido, pausas más largas

**Asunto del email con el video:**
- ES: "Grabé esto para [business_name] — míralo cuando puedas"
- EN: "I recorded this specifically for [business_name]"

**Plataforma:** Loom (loom.com) — gratis hasta 25 videos/mes
